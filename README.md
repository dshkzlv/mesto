**Описание**

Проект, в котором пользователь может редактировать данные профиля, ставить лайки фотографиям.

В проекте использованы:

Языки HTML, CSS и JS,
Организация файловой структуры по БЭМ Nested, 
Flexbox, 
Grid Layout, 
Адаптивная вёрстка.
